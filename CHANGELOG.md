 # Changelog
 
 ## [0.1.2] - 2026-01-05
 
### Improved
 
- Enhanced error message display with more detailed error information in UI
  - Network errors now show error code (e.g., "网络错误: -1009")
  - HTTP errors display the raw response body from server
  - Timeout errors show "请求超时"
  - Business errors display the server-returned message directly
